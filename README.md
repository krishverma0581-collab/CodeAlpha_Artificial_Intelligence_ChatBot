# 🤖 Artificial Intelligence Chatbot – Java |

This project is a Java-based AI chatbot.  
The chatbot uses Natural Language Processing (NLP) techniques and machine learning–style similarity scoring to interact with users.

## ✨ Features
- NLP text preprocessing & intent detection
- Automatic learning capability (learn new responses)
- Bag-of-words similarity scoring
- Swing GUI chat interface
- Persistent memory storage using learned.txt
- Multiple intents and dynamic responses

## 🛠 Technologies Used
Java, Maven, Swing GUI, NLP text processing, File handling, HashMap & Collections

## ▶ Run Instructions
Run the `ChatBotGUI.java` file from NetBeans or any IDE.

## 📌 Learning command example:
learn: what is ai | ai is artificial intelligence

## 💬 Example conversation
You: what is your name  
Bot: hii there, I am candis, your java learning chatbot.
